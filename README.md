# FIM
Member Forum
(Django) Forum For Institute Members

# Things Are Covered:
  * 2 types of user account
        * Administrator
        * General Member
  * Post option for Member
  * Taking Donation Through Online Payment
  * Social Authentication
  * Comment/thumbs up/ thumbs down/ optioin for each post.
  * Upload Media File
